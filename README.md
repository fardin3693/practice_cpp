# Practice C++

A collection of simple C++ programs for learning and practice.

## 🚀 Getting Started

### Prerequisites
You will need a C++ compiler installed on your system (e.g., GCC/G++ or Clang).

### How to Compile and Run
To compile a specific file, use the following command in your terminal:

```bash
g++ -o program_name filename.cpp
```

Example for `main.cpp`:
```bash
g++ main.cpp
./main
```

## 📂 Project Structure
- `main.cpp` - Main entry point and basic tests.
- `variable.cpp` - Practice with C++ variables and data types.

## 🛠️ Tools Used
- Language: C++
- Editor: Zed / VS Code
